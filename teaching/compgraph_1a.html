Graphics

The purpose of these two supervisions is to ensure you understand enough graphics to complete the Ticks. Ideally, we should have the first supervision between Ticks 1 and 2, and the second supervision between Ticks 2 and 3.

Please complete the following written exercises, and past paper questions. General instructions for handing in work can be found here. Answers should be concise. Please don't write more than half a page for the essay-style questions.

Supervision 1

  1. What are the ray parameters of the intersection points between ray (1,1,1) + t(−1,−1,−1) and the sphere centered at the origin with radius 1? [FCG p89]
  2. We can define a ray-tracing viewing pyramid with the following inputs:
    • o – the ray origin (or eye point),
    • u – the up vector,
    • l – the look vector,
    • d – the distance from the ray origin to the image plane,
    • w, h – the width and height of the image plane in world space, and
    • w_px, h_px – the width and height of the image plane in pixels.
    Sketch a diagram showing these variables. Give an expression for the ray cast through pixel (i,j) of the image plane. Assume the image origin is at the top left corner.
  3. Why do highlights on plastic objects look white, while those on gold metal look gold? How could you implement this with the Phong reflection model?[FCG p242]
  4. Explain how you could implement the following effects with distributed ray tracing:
    • Soft shadows cast by point lights,
    • Camera bokeh or depth-of-field, and
    • Anti-aliasing.
  5. If you have time, try the optional practical exercise – soft shadows and depth of field.

Supervision 2

  1. What is OpenGL? Why is it an API?
  2. What is an OpenGL shader?
  3. Sketch a diagram of the OpenGL rendering pipeline that includes the following stages:
    • Vertex Shader → Primitive Setup → Clipping → Rasterization → Fragment Shader
    Briefly explain what each stage does.
  4. What are the OpenGL Model, View, and Projection matrices?
  5. What is a vertex-array object? What is a vertex-buffer object?
  6. How do we use glDrawElements() to draw triangles?

Optional practical exercises

If you have time, I recommend you complete the optional exercises.

Tick 2b Distributed Ray Tracing

In this optional exercise, you will start to approach photo-realism. You will implement some distributed ray tracing effects: soft shadows from light sources and depth-of-field effects using the camera.

You should start from your completed Tick 2 code. You can use the following scene file to get the image on the left: tick2star.xml.

If you are eager explore ray tracing further, try implementing a Monte Carlo path tracer.

Textbooks

I recommend acquiring a copy of Fundamentals of Computer Graphics [FCG] – Shirley, P. & Marschner, S. (2009). This is the main textbook for the course and it covers the course's concepts and algorithms in detail, providing pseudo code and examples.

Some of the written supervision exercises are taken from the book.

Check for a copy in your college or the lab library, or try to find one online.

Another handy book is the OpenGL Programming Guide – John Kessenich, Graham Sellers, & Dave Shreiner. This book provides an excellent introduction to OpenGL, and will be helpful in completing the third practical exercise.

It is possible to find a copy of this book online.
