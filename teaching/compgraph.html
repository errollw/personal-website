Computer graphics & image processing

Work for this course includes:

  1. Written exercises – covering a range of course material
  2. Practical exercises – learn the algorithms better by coding them
  3. Past exam questions – discover what is expected in the exam

Submission of practical exercises

When you complete the practical work, you will have a Java class that implements Algorithm2Dso that it may work with Easel.

Please name your class in the form <crsid>_<ExerciseName>, e.g. the filenames for my supervision two solutions would be eww23_Bezier.javaand eww23_DouglasPeuker.java. Email me these Java source files.

Though it seems contrary to Java's object-orientated nature, the solution to each exercise is often best contained within one class only. This makes them more portable and easier to mark.

Textbook

I recommend Computer graphics: principles and practice – Foley, J.D., van Dam, A., Feiner, S.K. & Hughes, J.F. (1990) [FvDFH], as it covers the course's 2D rendering algorithms in detail, providing code and examples.

Also, used versions of the second edition can be purchased cheaply (< £5) online on Ebay or similar.

Supervision work

Please complete the following supervision exercises and past papers.

General instructions for handing in work can be found here.

I strongly encourage you to do as many practical exercises as possible, as algorithms are frequently examined and there is no better way to learn them than to code them yourself!

Supervision 1

This supervision concerns the background of the course, and some of the basic 2D line-drawing algorithms.

  • Practical exercises 1-1, 1-2, and 1-3 (8-octant midpoint line drawing)
  • Written exercises 1.1.1 to 1.1.4
  • Written exercises 1.2.1 to 1.2.4
  • Written exercise 1.3.1

Supervision 2

This supervision concerns Bézier curves (a popular exam topic) and other more advanced 2D graphical techniques.

  • Practical exercise 2-1 (adaptive Bézier drawing)
  • Practical exercise 2-2 (Douglas-Peuker line chain simplification)
  • Written exercises 2.1.1, 2.1.2, and 2.2.1
  • Written exercises 3.1.2, 3.1.3, and 3.1.4
  • 2009 Paper 4 Question 5

Supervision 3
