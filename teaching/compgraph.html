Submission of practical exercises

The supervision exercises require you to write a simple raytracer. You can use whatever programming language or tools for this as you wish, but I suggest using some that already have vector maths and the ability to set pixels in an image.

Therefore, I suggest that you write it in Javascript. You all should have an HTML5-compatible browser which can set pixels in a Canvas element, and I suggest using a library for vector maths.

I have packaged all you need in a .zip file that contains:

  • raytracer_week1.html – open this file in a web browser to run the Javascript and render the image.
  • raytracer_week1.js – this is a template for writing your own ray tracer.
  • vector.js – this is a vector maths library, no need to modify it.

The main file you should modify is raytracer_week1.js. Please rename this to "CRSID_raytracer_week1.js" with your appropriate CRSid and email it to me. Beware, if you modify this filename without changing it in raytracer_week1.html, it will stop working.

Textbook

I recommend acquiring a copy of Fundamentals of Computer Graphics [FCG] – Shirley, P. & Marschner, S. (2009). This is the main textbook for the course and it covers the course's concepts and algorithms in detail, providing pseudo code and examples.

Some of the supervision exercises are taken from the book exercises.

Check for a copy in your college or the lab library, or try to find one online.

Supervision 1

This supervision concerns the fundamental concepts of images and vision, simple raytracing, and shading.

Written exercises – Introduction and simple rendering

  1. Why is it better to look at faint stars and comets slightly off-centre rather than looking directly at them?
  2. Calculate the ultimate monitor resolution (i.e. colour pixels/inch) beyond which better resolution will be unnecessary.
  3. Why do we use square pixels? Would hexagonal pixels be better? What about triangles? Do you see any difficulties building graphics hardware with these other two schemes?

Written exercises – from the textbook

  1. What are the ray parameters of the intersection points between ray (1,1,1) + t(−1,−1,−1) and the sphere centered at the origin with radius 1? [FCG p89]
  2. Why do most highlights on plastic objects look white, while those on gold metal look gold? [FCG p242]

Practical work

Write a very simple ray-tracer. This will involve ray tracing a sphere and shading it with Phong illumination. If you are able, add more objects like blocks, and perhaps shadows and reflection.
