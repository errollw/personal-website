Practical exercises

I strongly encourage you to do these exercises, as algorithms are frequently examined and there is no better way to learn them than to code them yourself. If you struggle with the code, please let me know ahead of the supervision.

1 Introduction to Ray Tracing

In this exercise you will write code for a simple ray tracer. I suggest you use Java, and start with the following files:

  • tick1.zip – The source code that you will modify.
  • setup_t1t2.pdf – A worksheet explaining how to setup the code.
  • tick1.pdf – A worksheet explaining what to do.

The textbook section on Ray Tracing should be helpful [FCG p69].

If you are feeling more adventurous, I encourage you to write the ray tracer from scratch in the language of your choice. It should be able to handle multiple spheres and the Phong reflection model.

2a Shadows and Reflections

In this exercise you will extend the ray tracer to handle planes, multiple lights, shadows, and reflections. The files you will use are:

  • tick2.zip – The source code that you will modify.
  • tick2.pdf – A worksheet explaining what to do.

As before, feel free to use a different language. If you have time, extend the ray tracer to handle triangles.

2b Distributed Ray Tracing

In this optional exercise, you will start to approach photo-realism. You will implement some distributed ray tracing effects: soft shadows from light sources and depth-of-field effects using the camera.

You should start from your completed exercise 2a code. You can use the following scene file to get the image on the left: tick2star.xml.

If you are eager explore ray tracing further, try implementing a Monte Carlo path tracer.

3 Rasterization with OpenGL

In this exercise you will write code for rendering an image with OpenGL. The files you will use are:

  • tick3.zip – The source code that you will modify.
  • setup_t3.pdf – A worksheet explaining how to setup the code.
  • tick3.pdf – A worksheet explaining what to do.

The textbook on OpenGL should be helpful.

Textbooks

I recommend acquiring a copy of Fundamentals of Computer Graphics [FCG] – Shirley, P. & Marschner, S. (2009). This is the main textbook for the course and it covers the course's concepts and algorithms in detail, providing pseudo code and examples.

Some of the written supervision exercises are taken from the book.

Check for a copy in your college or the lab library, or try to find one online.

Another handy book is the OpenGL Programming Guide. This book provides an excellent introduction to OpenGL, and will be helpful in completing the third practical exercise.
